# 微信小程序接入说明

小程序源码位于 `miniprogram/`，目标是让已开通 online 的用户在手机端绑定手机号，并下载保存 online 生成的素材。

## 前置条件

1. 微信公众平台注册小程序，拿到 AppID 与 AppSecret。
2. 小程序后台开通手机号快速验证组件。
3. 小程序后台把云端 API 域名加入 request/downloadFile 合法域名，例如：
   - `https://bhzn.top`
   - 如素材后续改走独立 CDN，也要把 CDN 域名加入 downloadFile 合法域名。
4. 服务器 `.env` 配置：

```env
WECHAT_APP_ID=小程序AppID
WECHAT_APP_SECRET=小程序AppSecret
PUBLIC_BASE_URL=https://bhzn.top
```

改完后重启 `lobster_server`。

## 小程序开发配置

1. 用微信开发者工具打开 `lobster_server/miniprogram`。
2. 在微信开发者工具里把 `project.config.json` 的 `appid` 改成正式 AppID，或创建本机私有配置 `project.private.config.json`。
3. 如服务器域名不是 `https://bhzn.top`，修改 `miniprogram/config.js`：

```js
module.exports = {
  API_BASE: "https://你的服务器域名",
  APP_NAME: "必火AI员工"
};
```

## 业务流程

1. 小程序启动后调用 `wx.login`，请求 `POST /api/mobile/wechat-login`。
2. 如果微信 openid 已经绑定到平台手机号账号，服务端直接返回手机号账号 token。
3. 如果还没绑定，用户点击“授权手机号”，小程序拿到微信手机号 `code`，请求 `POST /api/mobile/devices/bind`。
4. 服务端校验手机号是否已在平台内注册：
   - 已注册：绑定手机设备与 online 账号，返回账号 token。
   - 未注册：返回“该手机号未在平台内注册过，没有 online 版本”。
5. 素材页请求 `GET /api/mobile/downloads?device_id=...`，返回可预览和可下载链接。
6. 小程序通过 `wx.downloadFile` 下载云端代理链接，再调用 `wx.saveImageToPhotosAlbum` 或 `wx.saveVideoToPhotosAlbum` 保存到相册。

## 相关接口

| 接口 | 说明 |
|------|------|
| `GET /api/mobile/phone/status?phone=手机号` | 检查手机号是否已有 online 账号。 |
| `POST /api/mobile/wechat-login` | 小程序 `wx.login` code 换 token。 |
| `POST /api/mobile/devices/bind` | 使用微信手机号授权 code 绑定手机号账号与手机设备。 |
| `GET /api/mobile/devices` | 查看当前账号的 online 设备与手机设备。 |
| `POST /api/mobile/devices/{device_id}/heartbeat` | 刷新手机设备在线状态。 |
| `GET /api/mobile/downloads?device_id=...` | 获取素材下载列表。 |

## 测试要点

- 手机号未注册时，绑定必须失败并提示没有 online 版本。
- 已注册手机号授权后，再次进入小程序应直接识别为已绑定。
- 素材列表至少覆盖 `assets.source_url`、定时任务结果、H5 会话结果中的视频/图片链接。
- 微信开发者工具里 request/downloadFile 域名校验要打开；若下载失败，优先检查合法域名和链接是否 HTTPS。
